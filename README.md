  - Person : List + CRUD

  - Status/Action:

    - Projet/Client of the Mission
    - Player = Actor

    - Priority

  - Person

    - Name
    - Exp
    - Availibity date

    - Price Category
    - Person Type
    - Sites OK
    - Sites KO

  - Person Type

  - Price Category

  - Skill

  - Site

  - Project
    - Name : example: "ProjectABC", "ProjectDEF", "H IJ", "ClientK", "SociétéL" ...
  
  - Player
    - Name
    - FirstName
  
  - Priority
